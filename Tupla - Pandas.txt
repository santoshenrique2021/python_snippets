#Definição - sequências imutáveis de armazenamento de ítens que podem ser heterogêneos.
##Similar a uma lista.

#Nota - a tupla é definida entre parênteses

tupla_v1 = tuple(['ELEMENTO_1','ELEMENTO_2', NUMERO, ...])

#Tipo de variável
type(tupla_v1) #tuple

#Acessar elementos de uma tupla
##Nota - python é indexado a partir de 0

tupla_v1[0] #ver o primeiro elemento
tupla_v1[1] #ver o segundo elemento
tupla_v1[-1] #ver o último elemento 
tupla_v1[1:3] #ver o primeiro e o segundo elemento/ slice
##Ponto de partida é incluído
##Ponto de parada não é incluído
